# 风眼 · Typhoon Eye

> 台风实况信息 + 分级应急预案，一页看懂、照做即可。极简、零依赖的静态单页，与 [蛛网之上](https://github.com/Mr-Salticidae/above-the-web) 同源的纸墨设计体系。

**⚠️ 当前为 v0.1 demo，使用内置演示数据。防灾决策请以当地政府与气象部门发布的官方预警为准。**

## 功能

- 🌀 **实况面板** — 台风位置、风力、气压、移向移速、风圈半径
- 🗺️ **路径示意图** — SVG 历史/预报路径，逐点悬停查看强度
- 🚦 **四级预警** — 蓝/黄/橙/红切换，颜色对齐官方预警信号
- ✅ **分级预案** — 每级一份可勾选的行动清单，进度自动保存
- ☎️ **应急信息** — 求助电话与权威信息源直达
- 🌗 **昼夜双主题**

## 使用

无需构建，克隆后直接双击 `index.html`，或：

```bash
npx serve .
```
